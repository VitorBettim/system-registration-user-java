# Sistema de Cadastro de Usuários 👤

Projeto desenvolvido em **Java** com o objetivo de praticar **Programação Orientada a Objetos (POO)** e manipulação de listas utilizando **ArrayList**.

O sistema permite realizar operações básicas de gerenciamento de usuários através de um **menu interativo no console**.

---

## 📌 Funcionalidades

O sistema possui as seguintes funcionalidades:

* ✅ Cadastrar usuário
* 📋 Listar usuários cadastrados
* 🔎 Buscar usuário pelo ID
* ❌ Remover usuário
* 🚪 Encerrar o sistema

---

## 🧠 Conceitos aplicados

Durante o desenvolvimento deste projeto foram utilizados conceitos importantes da programação:

* Programação Orientada a Objetos (POO)
* Classes e Objetos
* Encapsulamento
* Métodos
* Construtores
* ArrayList
* Estrutura de repetição (`while`)
* Estrutura de decisão (`switch`)
* Entrada de dados com `Scanner`

---

## 🗂 Estrutura do projeto

O projeto é composto pelas seguintes classes:

Usuario.java

Responsável por representar o usuário do sistema.

Atributos principais:

* id
* nome
* email

---

SistemaCadastro.java

Responsável por gerenciar os usuários do sistema.

Funções principais:

* cadastrarUsuario()
* listarUsuarios()
* buscarUsuario()
* removerUsuario()

---

Main.java

Classe principal do sistema.

Responsável por:

* exibir o menu
* receber dados do usuário
* chamar as funções do sistema
